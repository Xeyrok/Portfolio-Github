Création d'un Puzzle Tetris

Accueil

Description du Projet

Ce projet consiste à développer un puzzle basé sur des polyominos, inspiré du célèbre jeu Tetris. Le but est d'assembler ces formes pour minimiser l'espace occupé. Réalisé en groupe de quatre, le projet suit le design pattern MVC pour organiser le code.

Pour ma part, j'ai contribué à ce projet en travaillant sur la conception des modèles de pièces ainsi que sur la gestion de leur placement et de leurs interactions sur le plateau de jeu.

Fonctionnalités

  • Interface graphique pour le jeu
  • Gestion des pièces de puzzle et du plateau
  • Fonctionnalités pour ajouter, valider et supprimer des pièces
  • Algorithme de détection de collision et gestion des tests

Architecture du Code

  • Modèle : Représentation des pièces et du plateau de jeu
  • Vue : Interface utilisateur pour visualiser et interagir avec le jeu
  • Contrôleur : Gestion des événements et logique du jeu

Technologies Utilisées

  • Langage : Java
  • Bibliothèque : javax.swing, java.awt.event
  • Outil : Apache Ant pour l'automatisation des tâches

Conclusion

Le projet a permis de mettre en pratique le pattern MVC et d'améliorer nos compétences en développement Java. Il a également souligné l'importance des tests automatisés pour assurer la fiabilité du code.
