Création d'une bataille navale en Java

ACCUEIL

Description du Projet

Développer un jeu de bataille navale en Java où deux joueurs, l'un humain et l'autre contrôlé par un algorithme, s'affrontent sur une grille. Le but est de couler tous les navires de l'adversaire en choisissant des cases pour tirer. Ce projet s'est effectué en groupe de 4.

Pour ma part, je me suis occupé de la partie gestion des joueurs ainsi que de la partie CORE qui gère toute la logique du jeu.

Fonctionnalités

  • Gestion des positions des navires
  • Logique de jeu pour les tirs et les résultats
  • Interface utilisateur graphique (GUI) et en ligne de commande (CLI)
  • Mode de jeu contre un adversaire aléatoire

Technologies Utilisées

  • Langage : Java
  • Bibliothèques : Java Swing (interface graphique), Java AWT (composants d'affichage)
  • Architecture : Modèle-Vue-Contrôleur (MVC)

Architecture du Code

  • CORE : Classes de base pour la logique du jeu (navires, grille, cellules)
  • PLAYERS : Types de joueurs (HUMAN, RANDOMPLAYER)
  • INTERFACES : Versions CLI et GUI, gestion des entrées et sorties du jeu

Interfaces

GUI : Mode par défaut pour lancer depuis le terminal.
CLI : Mode accessible utilisant une représentation textuelle.

Gestion des Joueurs

  • Human : Interaction via l'interface
  • RandomPlayer : Joueur qui choisit des coups de manière aléatoire avec optimisation

Conclusion

Projet ayant approfondi les compétences en Java et la maîtrise du modèle MVC. Renforcement des compétences en conception d'applications et gestion de projets logiciels.
